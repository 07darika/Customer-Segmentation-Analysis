# Customer-Segmentation-Analysis

## Project Overview
This project performs customer segmentation based on music listening behavior using Power BI.

The objective is to identify customer groups based on behavioral and preference characteristics.

## Tools Used
- Microsoft Power BI
- DAX
- Data Visualization

## Dataset Features
- Danceability
- Energy
- Popularity
- Acousticness
- Valence
- Tempo
- Instrumentalness

## Customer Segments Identified

### Chill Acoustic Listeners
Prefer calm and acoustic music.

### High Energy Listeners
Prefer upbeat and energetic tracks.

### Mainstream Hit Followers
Prefer highly popular songs.

### Experimental Listeners
Show diverse niche listening patterns.

## Key Insights
- Distinct customer behavior patterns were identified
- Segment-wise preference analysis supports targeted recommendations
- Enables better personalization strategies

## Outcome
This project demonstrates practical experience in:
- Customer analytics
- Customer segmentation
- Behavioral analysis
- Dashboard development
